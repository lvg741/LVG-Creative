# LVG Creative Website

Deploy-ready Next.js website for LVG Creative.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. In Vercel, click Add New > Project.
3. Import the GitHub repository.
4. Click Deploy.
5. Add `www.lvgcreative.co.uk` in Project Settings > Domains.

## Notes

- The contact form currently uses `mailto:leah@lvgcreative.co.uk`.
- For a professional form, connect Formspree, EmailJS, Resend, or another form backend.
- Replace portfolio, social embed, booking, and testimonial placeholders with real content.
